# SimpleHttpServer in C++
This is a simple http server that was made for thesis work.
It is not production level and should not be treated as such.
The server is mainly for testing and experimenting

# Compiling
Make sure you are in the projects root folder and then run the commands bellow
```
mkdir build
cd build
cmake ..
cmake --build .
```

# Running the server
Make sure you are in the build folder and then run the command bellow
```
./server
```
